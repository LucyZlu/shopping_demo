# shopping_demo
一个例子入门Vue2.X+vue-router+Vuex+Webpack单页面应用程序
-
demo: https://lucyzlu.github.io/shopping_demo/index.html

技术栈：Vue2+vue-router+Vuex+webpack+ ES6/7<br>
```
git clone git@github.com:LucyZlu/shopping_demo.git

cd shopping_demo

npm install

npm run dev
```
